# dotfiles

## Dependencies

```bash
~$ docker pull initbar/lib:latest
```

## Setup

```bash
~$ curl https://raw.githubusercontent.com/initbar/dotfiles/main/scripts/install.sh | bash
~$ curl https://raw.githubusercontent.com/initbar/dotfiles/main/platforms/linux.sh | bash
```
